# SCT_ML_01
# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts house prices using Machine Learning based on different house features such as overall quality, living area, garage capacity, number of bedrooms, bathrooms, and year built.

The model is trained on the Ames Housing Dataset and demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and prediction.

---

## 🎯 Objective
- Predict house prices accurately.
- Analyze the factors affecting house prices.
- Build and evaluate a regression model.

---

## 📂 Dataset
**Dataset:** Ames Housing Dataset

Features include:
- Overall Quality
- GrLivArea
- Garage Cars
- Total Basement Area
- Full Bathrooms
- Bedrooms
- Year Built
- Lot Area
- Sale Price (Target)

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📊 Exploratory Data Analysis
The project includes:
- Missing value analysis
- Correlation heatmap
- Distribution plots
- Scatter plots
- Feature importance analysis

---

## 🤖 Machine Learning Model
Model Used:
- Linear Regression

Steps:
1. Data Loading
2. Data Cleaning
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Prediction
7. Model Evaluation

---

## 📈 Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## 📷 Results
The model successfully predicts house prices with good accuracy and visualizes relationships between important features using graphs and plots.
